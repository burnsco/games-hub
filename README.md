# Games Hub

Games Hub is a browser arcade built with Vite and React. It collects lightweight casual games behind a fast, no-account, instant-play interface.

## Highlights

- Multiple small games in one app
- Responsive layout for desktop and mobile play
- Lightweight UI built for quick session starts

## Stack

- Vite
- React
- TypeScript
- React Router

## Quick start

```bash
bun install
bun run dev
```

Build and deploy:

```bash
bun run build
bun run deploy
```
